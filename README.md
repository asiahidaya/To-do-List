# 📝 To-Do List Web App

A simple and lightweight To-Do List web application built using **HTML, CSS, and JavaScript**.
This project allows users to add and manage daily tasks efficiently.

---

## 🚀 Features

* Add new tasks
* Display task list dynamically
* Clean and minimal UI
* Responsive design (mobile-friendly)
* Uses JavaScript for DOM manipulation

---

## 📌 How It Works

1. User enters a task in the input field
2. Clicks the **Add** button
3. JavaScript captures the input
4. Task is added to the list (`<ul>`) dynamically
